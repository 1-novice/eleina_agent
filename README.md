# Eleina Agent - 全能智能体动漫角色

## 项目简介

Eleina Agent是一个基于本地大模型的智能体动漫角色系统，具有强大的推理能力、记忆系统和工具调用能力。系统采用模块化设计，支持多种模型接入方式，可通过本地API或直接加载本地模型进行推理。

## 核心功能

### 1. Agent 大脑层

- **大模型调度引擎**：支持本地模型和本地API两种模式，实现模型路由、降级和容错
- **意图识别与任务解析**：自动识别用户意图，抽取槽位信息，判断任务类型（扩展方向3层路由规则->小模型->大模型）
- **决策推理引擎**：实现思考链（CoT）和ReAct模式，支持工具选择和任务规划
- **执行控制引擎**：管理多步执行流程，处理上下文裁剪和结果整合

### 2. 记忆系统

- **短期记忆**：管理对话历史，实现上下文压缩和摘要
- **中期记忆**：保存用户偏好和历史任务进度
- **长期记忆**：存储用户档案和实体记忆，支持向量库和图谱（Neo4j）
- 存储优化方向:加**过期时间自动设置,加**Redis 连接池（高并发优化),加**记忆持久化到数据库**（重启不丢失）,单例模式（全局唯一）

### 3. 工具调用系统

- **工具注册中心**：管理工具元数据、权限和限流
- **工具执行引擎**：支持函数调用、API调用和代码执行
- **工具结果解析**：将结构化结果转换为自然语言

### 4. RAG 知识库

- **文档处理流水线**：支持上传、解析、清洗和分块
- **Embedding 向量化**：将文本转换为向量表示
- **向量数据库**：支持ChromaDB、Milvus等
- **召回与重排**：提高检索准确性

### 5. 多轮对话与状态管理

- **会话管理**：管理会话生命周期，支持中断恢复
- **状态机/工作流**：跟踪任务状态，支持复杂流程
- **多角色协同**：实现Supervisor调度和Worker分工

### 6. 输出与交互层

- **回复生成**：支持结构化输出和流式输出
- **多端接入**：提供CLI和API接口

### 7. 安全与合规

- **内容安全**：检测敏感词和恶意指令
- **数据合规**：实现数据脱敏和隐私保护
- **权限系统**：管理用户/角色/工具权限
- **审计日志**：记录全链路操作

### 8. 监控与运维

- **链路追踪**：集成langsmith和Jaeger
- **指标监控**：跟踪响应时长、成功率等
- **日志系统**：记录系统运行状态
- **告警机制**：及时发现和处理异常

### 9. 后台管理系统

- **用户管理**：管理用户账号和权限
- **知识库管理**：管理知识库内容和版本
- **Agent配置**：编辑提示词和角色配置
- **数据看板**：展示系统运行数据
- **对话回放**：查看历史对话记录

### 10. 自我进化模块

- **监控与评测**：监控系统性能，评测模型输出质量
- **诊断与优化**：识别问题并自动优化
- **离线运行**：不影响线上响应速度

## 安装与部署

### 环境要求

- Python 3.9+
- CUDA 11.7+ (推荐，用于加速模型推理)
- 足够的内存和存储空间（本地模型需要）

### 安装步骤

1. **克隆项目**
   ```bash
   git clone <repository-url>
   cd eleina-agent
   ```
2. **创建虚拟环境**
   ```bash
   python -m venv venv
   ```
3. **激活虚拟环境**
   - Windows:
     ```bash
     venv\Scripts\activate
     ```
   - Linux/MacOS:
     ```bash
     source venv/bin/activate
     ```
4. **安装依赖**
   ```bash
   pip install -r requirements.txt
   ```
   或使用Poetry:
   ```bash
   poetry install
   ```
5. **配置环境变量**
   编辑 `.env` 文件，设置模型路径、API配置等：
   ```env
   # Model Configuration
   MODEL_TYPE=local_api  # 可选: local, local_api
   LOCAL_MODEL_PATH="/path/to/qwen2.5-7B-instruct"
   LORA_PATH="/path/to/lora/fine-tuned"
   LOCAL_API_URL=http://localhost:8000/v1/chat/completions
   ```

## 使用方法

### 启动系统

```bash
python main.py
```

### 与Agent交互

系统启动后，您可以通过命令行与Agent进行交互：

```
=== 全能智能体动漫角色Agent ===
输入 'exit' 退出
用户: 介绍一下你自己
Agent: 我是一个全能的智能体动漫角色，具有丰富的知识和强大的能力。我可以帮助你解决各种问题，提供准确、有用的信息和建议。
```

### API接口

系统提供RESTful API接口，可通过以下方式访问：

```bash
http://localhost:8000/api/v1/chat
```

## 配置说明

### 模型配置

- **local模式**：直接加载本地模型文件
- **local\_api模式**：通过本地API服务进行推理

### 提示词配置

系统提示词存储在 `src/prompt/system_prompt.txt` 文件中，可根据需要修改。

### 工具配置

工具配置存储在 `.env` 文件中，可通过环境变量启用或禁用特定工具。

## 故障排除

### 常见问题

1. **模型初始化失败**：检查模型路径是否正确，CUDA是否可用
2. **API调用失败**：检查API服务是否运行，URL是否正确
3. **内存不足**：考虑使用4bit量化或切换到local\_api模式

### 日志查看

系统日志会输出到控制台，可通过调整 `.env` 文件中的 `LOG_LEVEL` 来控制日志级别。
